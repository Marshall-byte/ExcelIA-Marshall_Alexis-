# ExcelIA-Marshall_Alexis-
Repositorio en el que se encuentra los archivos para el proyecto de EXCEL con IA de Eanx
"Documentación del proceso
Tema elegido: Análisis y visualización de ventas mensuales de una panadería pequeña.

Fuente de datos: Base de datos generada de forma sintética (ficticia) mediante IA, estructurada con columnas de fecha, producto, categoría, cantidad y precio para su análisis.

Funciones o herramientas de IA usadas: * Generación de estructura de datos y contenido mediante IA (Gemini).

Asesoría en fórmulas de cálculo (=PRODUCTO(cantidad, precio)).

Guía paso a paso para la creación de Tablas Dinámicas.

Instrucciones para la creación de gráficos de torta y visualización de datos.

Conclusiones del análisis: * La herramienta permitió organizar y estructurar datos de manera eficiente para detectar tendencias.

El uso de tablas dinámicas facilita la comparación de rendimiento por categorías (Bebidas, Panadería, Almuerzo) en diferentes meses.

La visualización (gráfico de torta) permite identificar rápidamente qué categoría aporta el mayor porcentaje a los ingresos totales, facilitando la toma de decisiones estratégicas, como promociones o ajustes en el inventario."
